◆ translateCamera()

void de.emm.teama.chibaapp.Model3D.entities.Camera.translateCamera ( float  dX,
float  dY 
)

Translation is the movement that makes the Earth around the Sun. So in this context, translating the camera means moving the camera around the Zero (0,0,0)

This implementation makes uses of 3D Vectors Algebra.

The idea behind this implementation is to translate the 2D user vectors (the line in the screen) with the 3D equivalents.

In order to to that, we need to calculate the Right and Arriba vectors so we have a match for user 2D vector.

Parameters
dXthe X component of the user 2D vector, that is, a value between [-1,1]
dYthe Y component of the user 2D vector, that is, a value between [-1,1]
